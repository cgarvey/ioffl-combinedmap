IrelandOffline Broadband Map

A Google Maps / Fusion Tables implementation showing broadband services in Ireland used on irelandoffline.org.

Download .zip Download .tar.gz View on GitHub

About

This is a Google Maps implementation using Google Fusion Tables to display a variety of broadband, and broadband-related, services in Ireland. It is used by IrelandOffline.
